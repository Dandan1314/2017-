Array类型 创建数组有两种方法: new Array(); []; new Array()可以规定数组的长度; 数组的length不是只读的,可以修改 通过length我们可以很方便的给数组添加值,array[array.length] = "xiaoming"; 检测数组 使用Array.isArray();(ie9 ff4+ safari5+ opera10.5+); 转换方法; toString toLocalString valueof: 返回以逗号隔开的数组中的值; join() :以传入的字符分割数组; 如果数组中有undefined或null,使用以上方法会返回""; 栈方法(后进先出) pop()在数组末尾删除 // 返回被删除的; push();在数组末尾添加 // 返回新数组的长度; 队列方法(先进先出) shift:在数组开头删除 //返回被删除的; unshift:在数组开头添加 // ie7及老版本返回undefined;返回新数组的长度; 重排序方法 sort();//正排序 调用toString比较字符串排序,所以不准确 在sort方法中A在B前面返回-1;在后面返回1;相等返回0;所以可以编辑一个参数传入sort方法中,这样可以得到我们想要的结果 reverse();//反排序 以上两种方法返回新数组; 操作方法 concat合并数组 slice截取数组 splice:删除:传入两个参数,第一个参数为起始位置,第二个为要删除的个数 插入:传入大于两个参数,起始位置,0,要插入的值,.... 替换:传入大于两个参数,起始位置,要替换几个值,要替换的新值; splice会返回一个操作的新数组; 位置方法(ie9+) indexOf 有返回索引,无返回-1 lastIndexOf :有返回倒向索引,无返回-1; 迭代方法 every();//如果数组每项都返回true,则返回true; some(); //如果数组中有一项返回true,则返回true; foreach(); //同for循环,没有返回值 map();//执行操作后,返回操作后的数组 filter();//过滤数组,返回过滤后的数组 缩小方法 reduce(); reduceRight(); 可以给数组中的数求和;
